Dogs vs Cats Classifier

This project uses concepts in machine learning to classify dogs and cats. The model used was a multi-layer convolutional neural network using Tensorflow and Keras. The graphical user interface was built with Tkinter. The accuracy of this model turned out to be 95.65%.
